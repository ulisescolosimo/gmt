# Assets

## Logo

Coloca el archivo `logo.png` en esta carpeta.

**Especificaciones recomendadas:**
- Tamaño: 200x200px (mínimo 100x100px)
- Formato: PNG con transparencia
- Nombre: `logo.png`

El logo se mostrará en:
- Header (40x40px)
- Footer

## Imágenes adicionales (opcional)

Si deseas agregar más imágenes para el hero o otras secciones, puedes colocarlas aquí y referenciarlas en el HTML/CSS.
